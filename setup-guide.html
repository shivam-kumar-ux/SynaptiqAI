SYNAPTIQ SETUP GUIDE

Secure Gemini deployment with Firebase, GitHub, and Vercel
01
CONFIGURE FIREBASE
Authentication and Firestore
  1. Create a Firebase project and add a web app.
  2. Paste the generated config into js/firebase-config.js.
  3. Enable Email/Password and Google sign-in.
  4. Create Firestore and apply rules before production use.
Firebase web config is safe to expose publicly. It is not the Gemini secret.
02
KEEP GEMINI SERVER-SIDE
No browser access to the API key
  1. Create a Gemini API key in Google AI Studio.
  2. Never paste it into HTML, CSS, or frontend JavaScript.
  3. Never store it in localStorage.
  4. Set it only as a hosting environment variable named GEMINI_API_KEY.
SYNAPTIQ now uses only Gemini through a secure backend route.
03
PUSH THE REPO TO GITHUB
Source control and hosting entry point
  1. Create a GitHub repository.
  2. Push this project to that repository.
  3. Use GitHub for version control and collaboration.
04
DEPLOY WITH VERCEL
Secure hosting for the frontend and API route
  1. Import the GitHub repo into Vercel.
  2. Add the environment variable GEMINI_API_KEY in Vercel project settings.
  3. Deploy the project.
  4. Add the Vercel domain to Firebase authorized domains.
GitHub Pages is static-only, so it cannot protect a server-side Gemini secret. Hosting the code on GitHub is fine, but deploy the app with Vercel to keep the key private.
05
UNDERSTAND THE SECURE FLOW
Frontend and backend responsibilities
Browser UI -> /api/gemini -> Gemini API Frontend: - sends prompts - never stores the Gemini key - can be inspected safely Backend: - reads GEMINI_API_KEY from environment variables - calls Gemini - returns only model output
06
VERIFY BEFORE GOING LIVE
Quick checklist
  • Firebase config added
  • Authentication enabled
  • Firestore created
  • Repo pushed to GitHub
  • Project deployed on Vercel
  • GEMINI_API_KEY added in Vercel
  • Vercel domain added to Firebase
  • Settings page Gemini connection test passes
  • Plan generation, quiz, assessment, and report all work
